# 3D Landing Page (Three.js + WebGL)

Интерактивный 3D-лендинг на базе Three.js, демонстрирующий:
- рендеринг WebGL-сцены с анимациями,
- орбитальные контролы для обзора модели,
- параллакс-эффект при прокрутке,
- переключение темы (тёмная/светлая),
- простые метрики производительности (FPS, количество объектов).

Stack: Three.js + WebGL (+ Vite для локальной разработки)

## Возможности
- Анимированный 3D-объект (TorusKnot) с материалом и подсветкой.
- Группа мелких «орбов», двигающихся по окружности вокруг главного объекта.
- Звёздное поле в фоне (Points), создающее объёмную сцену.
- Орбитальные контролы (OrbitControls) с демпфированием.
- Параллакс на скролле: смещение позиции камеры по Y/Z.
- Переключатель темы (кнопка «Тема»).
- Кнопка CTA для перехода к информационному разделу.
- Адаптивность, обновление размера при ресайзе.

## Установка и запуск
Требуется Node.js >= 18.

```bash
npm install
npm run dev
```

Откройте выданный локальный URL в браузере. Для production-сборки:

```bash
npm run build
npm run preview
```

## Структура
- `index.html` — оболочка приложения, элементы UI.
- `src/main.js` — инициализация Three.js сцены, анимации и интерактив.
- `README.md` — описание проекта и инструкции.
- `ROADMAP.md`, `ROADMAP.en.md` — план развития на двух языках.
- `.gitignore` — служебные исключения, включая локальные файлы.

## Что решает
- Позволяет быстро создать красивую промо-страницу с 3D-графикой.
- Даёт основу для добавления моделей, шейдеров и эффектов.
- Обеспечивает минимальный UX (темы, CTA, метрики) и простую расширяемость.

## Технические замечания
- Используется `OrbitControls` из `three/examples/jsm`.
- Цветовое пространство рендера — `SRGBColorSpace`.
- Анимации рассчитаны по времени кадра; рендер — на `requestAnimationFrame`.
- Параллакс основан на текущем `scrollY` относительно высоты вьюпорта.
