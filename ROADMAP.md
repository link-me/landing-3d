# ROADMAP (RU)

План развития без привязки к датам:

Готово (основа):
- Строгое орбитальное движение пилюль вокруг ядра.
- Реалистичные импульсные столкновения между пилюлями, коррекция проникновения с итерациями.
- Демпфирование локальных смещений, ограничение свободных скоростей, настройка упругости и трения.

Дальше:
- Добавить UI-панель для настройки параметров физики (restitution/friction/damping/velocity cap).
- Ввести короткий всплеск яркости подсветки при столкновении (collison flare) с таймингом.
- Постобработка: Bloom/ToneMapping/FXAA с пресетами для разных тем.
- Интегрировать ShaderMaterial для фоновых эффектов (градиенты, шум, волны).
- Адаптация производительности: качество материалов/LOD на слабых устройствах.
- Триггеры по скроллу (scroll-driven): запуск анимаций и подсказок.
- Модуль трекинга ошибок/перформанса (Sentry/кастомный логгер).
- Сборка и деплой (static hosting/CI/CD), пресеты окружений.
- Темизация через CSS variables и пресеты материалов.
- Расширение UI: переключатели эффектов, пресеты сцен, быстрые ссылки.
