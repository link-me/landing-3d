# ROADMAP (RU)

План развития без привязки к датам:
- Добавить загрузку 3D-моделей (GLTF/GLB) и лёгкий ассет-менеджмент.
- Ввести анимации переходов между блоками лендинга (GSAP/Custom).
- Добавить постобработку (Bloom/ToneMapping/FXAA) с настраиваемыми пресетами.
- Интегрировать простые шейдерные эффекты (ShaderMaterial) для фоновых элементов.
- Реализовать адаптацию производительности на слабых устройствах (качество/LOD).
- Добавить интерактивные триггеры по скроллу (scroll-driven animation).
- Создать модуль для трекинга ошибок и перформанса (Sentry/Custom logger).
- Подготовить сборку и деплой (Static hosting/CI/CD).
- Ввести механизм темизации через CSS variables и пресеты материалов.
- Расширить UI: переключатели эффектов, пресеты сцен, быстрые ссылки.
